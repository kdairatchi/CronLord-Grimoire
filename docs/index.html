Add your ritual

Got a cron that's been quietly running your life for years? Package it as a ritual and send a PR.

  1. Fork kdairatchi/CronLord-Grimoire.
  2. Copy the closest existing ritual under rituals/<category>/ as a starting point.
  3. Keep enabled = false. Shell body must be /bin/sh-compatible. Secrets via env vars.
  4. Run python3 scripts/gen-index.py so INDEX.md + docs/rituals.json stay in sync.
  5. Open one PR per ritual. Header-comment pattern mandatory.
